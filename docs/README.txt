This is the docs folder, which contains all the documentation for `pkm-server`.

They're simple text files for the sake of ease (this really doesn't need formatting).

====================
Documentation TOC
====================
 `PUBLIC_DIRECTORY.txt` - Explains how packages are served.
 `CREDITS.txt` - The credits for this project.


====================
Important Notes
====================

Please take care with formatting of JSON files. Incorrect formatting will cause the Windows 96 package manager to ignore your repository!

You can use an editor like Geany/Notepad++/VSCode to help you correctly format them.
